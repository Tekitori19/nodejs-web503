Hướng dẫn sử dụng Tailwind CSS cho người mới bắt đầu

Nguyễn Văn A

15 Tháng 1, 2024 · 5 phút đọc

Tailwind CSS là một framework CSS utility-first giúp bạn xây dựng giao diện nhanh chóng và linh hoạt. Trong bài viết này, chúng ta sẽ tìm hiểu về cách cài đặt và sử dụng Tailwind CSS trong dự án của bạn.

1. Cài đặt Tailwind CSS

Để bắt đầu với Tailwind CSS, bạn cần thực hiện các bước sau:

npm install tailwindcss
                        npx tailwindcss init

2. Cấu hình cơ bản

Sau khi cài đặt, bạn cần tạo file cấu hình tailwind.config.js:

module.exports = {
                        content: ["./src/**/*.{html,js}"],
                        theme: {
                        extend: {},
                        },
                        plugins: [],
                        }
